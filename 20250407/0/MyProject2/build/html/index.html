Module restcalend¶

Модуль для генерации календаря в формате reST.

Основной функцией является restmonth, которая генерирует календарь для указанного месяца и года.

Для использования через командную строку используйте:

python3 -m restcalend ГОД МЕСЯЦ

Пример:

python3 -m restcalend 2025 4

Будет сгенерирован календарь для апреля 2025 года в файлы .txt и .rst.

Вся техническая документация доступна по ссылке:

https://example.com/tech-docs

Module restcalend.py¶

restcalend.restcalend.restmonth(year, month)¶

Генерирует календарь для указанного месяца и года в два файла: .txt и .rst.

Аргументы:

year (int): Год для календаря. month (int): Месяц для календаря.

Возвращает:

tuple: Пути к созданным файлам .txt и .rst.
